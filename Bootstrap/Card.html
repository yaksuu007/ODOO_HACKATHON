Class Utility Explanation
  • card: Main Bootstrap card container with padding and border
  • card-img-top: Image that fits and aligns at the top of the card
  • card-body: Section inside the card that holds title, text, and buttons
  • text-center: Aligns text to center
  • btn btn-primary / btn-success: Bootstrap styled buttons
  • mb-4 / mb-5: Margin bottom spacing (1.5rem, 3rem)
  • me-3: Margin end (right side) spacing between cards in a group
  • card-group: Wraps multiple cards into a responsive flex layout
