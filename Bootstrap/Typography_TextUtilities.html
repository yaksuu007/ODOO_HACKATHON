Typography in Bootstrap

fs-1 (Largest text size)

fs-3 (Medium-large)

fs-5 (Default size)

fs-6 (Smallest size)

fw-bold (Bold text)

fw-semibold (Semi-bold)

fw-normal (Regular weight)

fw-light (Light weight)

fw-lighter (Thinnest weight)

text-primary = Bootstrap blue

text-success = green

text-danger = red

text-muted = light gray (for less emphasis)

text-start = left aligned

text-center = center aligned

text-end = right aligned

.lead = larger font, lighter weight

small = smaller text, usually for disclaimers

text-uppercase = ALL CAPS

TEXT-LOWERCASE becomes lowercase

text-capitalize = First Letter Of Each Word Capitalized

font-monospace = fixed-width font (useful for code)

Bootstrap Spacing Utilities
  • mb-4: Margin bottom with spacing level 4
  • mt-3: Margin top with spacing level 3
  • ms-2: Margin start (left in LTR languages)
  • me-2: Margin end (right in LTR languages)
  • p-3: Padding on all sides
