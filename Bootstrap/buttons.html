Utility Classes Explanation
  • mb-2 / mb-3 / mb-4: Margin bottom (0 to 5 scale, each step = ~0.25rem)
  • me-2: Margin end (right in LTR) — separates buttons horizontally
  • mt-5: Top margin (used on the outer container to push down from top)
  • my-5: Margin top and bottom (used for vertical spacing around elements like `
    `)
  • w-100: Width 100% — button fills full row
  • btn-sm / btn-lg: Controls font size and padding for buttons
  • btn-outline-*: Creates bordered buttons without fill color
